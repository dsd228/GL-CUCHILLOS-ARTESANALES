📚 Cómo configurar GitHub Pages

Pasos para que "Publicar" actualice el sitio

  1. Creá un repositorio en GitHub (público)
  2. Subí los archivos: index.html, admin.html, producto.html y products.json
  3. Repo → Settings → Pages → Source: main
  4. Creá un token: Settings → Developer → Fine-grained tokens → Contents: Read & Write
  5. Pegá el token acá. El botón "Publicar" actualiza el sitio en ~30 seg.
