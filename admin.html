📚 Cómo configurar GitHub Pages

Seguí estos pasos para que "Publicar" actualice el sitio automáticamente

  1. Creá un repositorio en GitHub (público)
  2. Subí los 3 archivos: index.html, admin.html y products.json
  3. En el repo → Settings → Pages → Source: Deploy from branch → main
  4. Creá un token en: GitHub → Settings → Developer Settings → Fine-grained tokens → seleccionar el repo → Contents: Read & Write
  5. Pegá el token acá y guardá. Desde ahora el botón "Publicar" actualiza el sitio automáticamente en ~30 seg.

Tu sitio estará en: https://[usuario].github.io/[repo]/
