📚 Cómo configurar GitHub Pages

1. Creá un repositorio en GitHub (público o privado con Pages habilitado)

2. Subí los archivos: index.html, admin.html y products.json

3. En el repositorio → Settings → Pages → Source: Deploy from branch → main

4. Creá un token: GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained → seleccionar repo → Contents: Read & Write

5. Pegá el token en este panel y guardá. Desde ahora, "Publicar cambios" actualiza el sitio automáticamente.

⚡ Los cambios tardan ~30 segundos en verse reflejados en el sitio live.
